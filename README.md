# Fib's configs

## Requirements
- Arch Linux Distro
- Terminal
- Internet Connection
- Brain

## Installation
Run this command
```bash
curl -LO https://raw.githubusercontent.com/NoahPsight/.dotfiles/main/setup.sh
sh setup.sh
```
This command should download the setup script wherever you're CD'd and run it.

This repo is not extremely feature packed; It's existence is so when I inevitably blow up my system I can recreate the essentials.
