# Fib's configs

## Requirements
- Arch Linux Distro using pacman
- Terminal
- Internet Connection
- Brain

## Installation
Run this command
```bash
curl -L -o delete_me https://raw.githubusercontent.com/NoahPsight/.dotfiles/main/setup.sh
sh delete_me
```
This command should download the setup script wherever you're CD'd, run it, then delete it.

This repo is not extremely feature packed; It's existence is so when I inevitably blow up my system I can recreate the essentials.
